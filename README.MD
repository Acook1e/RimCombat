# Rim Combat

> 面向上古卷轴 5 SE/AE 的近战战斗重构原型。
>
> 当前版本的重点是把武器分类、攻击耐力、力竭、架势、韧性/硬直、格挡/限时格挡、战技和对应 UI 接成一条可以长期实测、持续调数值、继续扩展的近战循环。
>
> 本 README 的完成度评估默认不计处决系统。处决代码仍然保留在项目中，但因为 OAR 侧联动问题，目前暂时搁置。

更多说明见 [WIKI.MD](WIKI.MD) 和 [WIKI_ZH.MD](WIKI_ZH.MD)。

---

## 当前状态

- 当前阶段：Pre-Alpha，但已经不是“只有几个半成品模块”的状态。
- 当前结论：已经进入可以整体测试、调数值和查兼容的阶段。
- 当前最成熟的部分：武器分类、攻击耐力、力竭、格挡/限时格挡、基础架势链路、特殊空手/生物接口、玩家侧战技分配与 HUD。
- 当前仍在迭代的部分：架势崩溃反馈、特殊硬直表现与细则、战技内容量、NPC 侧战技支持。
- 当前不建议期待：正式发布级平衡、完整内容池、成熟的处决演出与镜头系统。

如果只用一句话概括现在的项目状态：

Rim Combat 的主战斗循环已经成型，而且已经有持续测试价值；但它现在仍然是“高可玩原型”，不是“收尾完成的正式 overhaul”。

## 开启后会发生什么

- 不同武器类型会读取不同的基础耐力、架势伤害、韧性伤害和格挡强度，而不是统一走原版的粗略规则。
- 攻击不再只是随便挥砍。普通攻击、重击、重量修正、Precision 命中和动画 payload 都会参与耐力消耗。
- 耐力见底和力竭会真正影响攻防节奏，NPC 也会受到同类限制。
- 格挡不再只是硬吃伤害，而是和耐力消耗、架势伤害、限时格挡窗口、反硬直收益一起工作。
- 受击层不再只有原版 stagger，而是变成“内部韧性阈值 + 特殊硬直扩展 + 可配置恢复时间”的组合。
- 战技可以按武器分配，支持准备态/启用态、法术派生、HUD 状态显示和背包内右半屏分配菜单。

## 完成度判断

下面的完成度按“当前是否接进主循环、是否能进游戏反复测、还差多少收尾”来估算，不是精确工程百分比。

| 范围 | 当前判断 |
| --- | --- |
| 核心近战循环（不计处决） | 约 80% 到 85%，已经进入联调与数值打磨阶段 |
| 玩家侧可用性 | 约 75% 到 80%，主要玩法入口已经具备 |
| 内容量与表现层 | 约 50% 到 60%，系统骨架先行，内容深度仍不足 |
| 当前是否适合测试 | 是，已经适合持续实机测试 |

当前最值得做的事情已经不是怀疑“能不能测”，而是：

1. 继续验证数值手感和节奏。
2. 补强架势崩溃与特殊硬直的表现反馈。
3. 扩充战技内容和 NPC 侧接入。

## 依赖与用户入口

### 推荐搭配

| 类型 | 说明 |
| --- | --- |
| 必需 | SKSE |
| 动画相关 | OAR、MCO 或 BFCO |
| UI 与反馈 | PrismaUI、TrueHUD |
| 硬直生态推荐 | Modern Stagger Lock |
| 可选强化 | OCF，可提供更细的武器分类 |

说明：

- 没有 PrismaUI 时，战技菜单与 HUD 无法正常工作。
- 没有 TrueHUD 时，架势条和部分反馈会不完整。
- 没有 Modern Stagger Lock 时，当前版本的硬直体验会明显不完整。
- 没有 OCF 时，项目仍可运行，但武器分类会退回较粗的原版判断。

### 当前操作入口

- 在背包菜单打开时，按 `T` 开关战技分配菜单。
- 默认按 `Left Alt` 切换战技状态。
- 战技菜单会跟随当前背包中选中的武器刷新。
- 仓库自带 Elden Rim 风格的样例战技，可直接用于系统链路测试。

### 当前配置文件

- `Data/SKSE/Plugins/RimCombat/Settings.json`
- `Data/SKSE/Plugins/RimCombat/SettingsDefault.json`
- `Data/SKSE/Plugins/RimCombat/WeaponArt/*.json`
- `Data/SKSE/Plugins/RimCombat/Stagger/*.json`
- `Data/SKSE/Plugins/RimCombat/Localization.json`

如果你现在的目标是试玩与调参，最值得优先观察的是：

1. 不同武器类型下的耐力成本与格挡强度是否符合预期。
2. 力竭、架势、格挡和硬直之间是否形成了清晰的攻防节奏。
3. 限时格挡是否有足够收益，但又不会强到压过其他选择。
4. 战技的切换、分配、法术派生与 HUD 提示是否顺畅。
5. 特殊硬直与外部硬直/韧性模组的衔接是否自然。

## 面向玩家的结论

如果你想要的是一套已经完全收尾、内容量充足、演出稳定的正式战斗 overhaul，Rim Combat 现在还没到那个阶段。

但如果你想测试一套更强调耐力承诺、力竭惩罚、架势压制、格挡博弈、战技切换和特殊硬直联动的近战原型，那么它现在已经值得进游戏认真测试。
